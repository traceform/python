**Question 2:** What is the output of the following snippet?

```python
tup = 1, 2, 3
a, b, c = tup

print(a * b * c)

```

<details>
  <summary>Check</summary>

  The program will print `6` to the screen. The `tup` tuple elements have been "unpacked" in the `a`, `b`, and `c` variables.
</details>
