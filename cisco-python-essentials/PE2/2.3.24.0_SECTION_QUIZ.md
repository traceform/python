**Question 1:** What is the expected output of the following code?

```
for ch in "abc123XYX":
    if ch.isupper():
        print(ch.lower(), end='')
    elif ch.islower():
        print(ch.upper(), end='')
    else:
        print(ch, end='')
```

<details>
  <summary>Check</summary>

  ```Output
  ABC123xyx
  ```
</details>
