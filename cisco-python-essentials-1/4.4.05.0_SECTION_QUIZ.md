**Question 1:** What is the output of the following snippet?

```python
def message():
    alt = 1
    print("Hello, World!")


print(alt)

```

<details>
  <summary>Check</summary>

  The `NameError` exception will be thrown:

  `NameError: name 'alt' is not defined`
</details>

**Question 2:** What is the output of the following snippet?

```python
a = 1


def fun():
    a = 2
    print(a)


fun()
print(a)

```

<details>
  <summary>Check</summary>

  ```Output
  2
  1
  ```
</details>

**Question 3:** What is the output of the following snippet?

```python
a = 1


def fun():
    global a
    a = 2
    print(a)


fun()
a = 3
print(a)

```

<details>
  <summary>Check</summary>

  ```Output
  2
  3
  ```
</details>
