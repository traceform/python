**Question 1:** What is the output of the following snippet?

```
x = 1
y = 0

z = ((x == y) and (x == y)) or not(x == y)
print(not(z))

```

<details>
  <summary>Check</summary>

  ```Output
  False
  ```
</details>

**Question 2:** What is the output of the following snippet?

```
x = 4
y = 1

a = x & y
b = x | y
c = ~x  # tricky!
d = x ^ 5
e = x >> 2
f = x << 2

print(a, b, c, d, e, f)

```

<details>
  <summary>Check</summary>
  
  ```Output
  0 5 -5 1 1 16
  ```
</details>
